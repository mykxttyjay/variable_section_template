## Maintenance Guidelines

### Adding New Pages

When adding new pages, follow these guidelines:

1. **Identify Page Type**

   - Service page: Focus on service-specific + location keywords
   - Location page: Focus on city-specific + service keywords
   - Content page: Focus on informational keywords
   - Utility page: Focus on navigational keywords

2. **Research Keywords**

   - Use Google Keyword Planner
   - Check competitor keywords
   - Analyze search intent
   - Verify search volume

3. **Assign 6-10 Keywords**

   - Include primary keyword (highest volume)
   - Include 2-3 secondary keywords
   - Include 3-5 long-tail keywords
   - Ensure uniqueness (check this map)

4. **Validate**
   - Run `node scripts/analyze-keywords.mjs`
   - Check for cannibalization
   - Verify keyword count
   - Update this map

### Updating Existing Keywords

1. Review current performance in Google Search Console
2. Identify underperforming keywords
3. Research better alternatives
4. Update MDX file metadata
5. Run validation script
6. Update this map
7. Monitor changes for 30 days
